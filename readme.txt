JHK Team Management lets you easily create, organize, and display team member profiles on your WordPress site in grid or list layouts.

== Description ==

The Team Management Plugin allows you to easily manage and display team members on your WordPress website. You can create custom profiles for each team member, categorize them by group, designation, or department, and display them using a shortcode. The plugin supports two layouts: grid and list, giving you flexibility in how you present your team.

== Features ==

- Custom Post Type: Create custom profiles for your team members.
- Shortcode Support: Display team members on any page or post using a simple shortcode.
- Grid and List Layouts: Choose between grid or list layout for displaying team members.
- Categorization: Organize team members by group, designation, or department.
- Profile Fields: Add name, image, post, short bio, email, phone number, experience, and CV/resume upload option for each team member.
- Social Media Links: Include Facebook, Instagram, LinkedIn, Twitter, and YouTube links for each team member.
- Popup Display: Additional information displayed in a popup window when clicking on a team member's profile.

== Installation ==

1. Upload the 'jhk-team-management' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create new team members under 'Team' in the WordPress admin dashboard.
4. Use the provided shortcode to display team members on your website.

== Frequently Asked Questions ==


= How do I add team members to multiple groups or departments? =

You can assign team members to multiple groups or departments by using categories or tags functionality provided by WordPress.

= Can I display team members from a specific group or department using the shortcode? =

Yes, you can use attributes in the shortcode to specify which group or department of team members to display.

= Can team members be sorted alphabetically by name? =

Yes, you can sort team members alphabetically by name using options available in the shortcode parameters.
